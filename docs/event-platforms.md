# イベント告知・集客・参加者管理ツール比較

Luma / connpass のようなイベント管理ツールの特徴比較。

## 1. モダン・コミュニティ重視（Luma 系）

| ツール | 特徴 |
|--------|------|
| **Doorkeeper** | コミュニティ（グループ）軸。リピーター育成に強い。勉強会向け。 |
| **Partiful** | スマホでオシャレな招待ページ。カジュアルな交流会・ミートアップ向け。 |

## 2. IT・エンジニア特化（connpass 系）

| ツール | 特徴 |
|--------|------|
| **TECH PLAY** | エンジニア・デザイナー層が多い。企業採用系イベント・ハッカソン向け。 |

## 3. 万能型・一般向け

| ツール | 特徴 |
|--------|------|
| **Peatix** | 国内最大。有料チケット決済・QR受付。幅広いジャンル。 |
| **こくちーずプロ** | SEO 強い。無料機能充実。セミナー・サロン向け。 |

## 4. BtoB・ビジネス向け

| ツール | 特徴 |
|--------|------|
| **EventHub / Eventory** | 参加者マッチング・商談・動画配信・アンケート一元管理。企業主催向け。 |

## 選び方

| 目的 | おすすめ |
|------|---------|
| デザイン・コミュニティ重視（Luma 代替） | Doorkeeper, Partiful |
| エンジニア集客（connpass 代替） | TECH PLAY |
| 一般・趣味・有料チケット販売 | Peatix |
| 会社のビジネスセミナー | EventHub, こくちーずプロ |
